스프링 프레임워크

핵심 기술: 스프링 DI 컨테이너,AOP,이벤트
웹기술 스프링 MVC

스프링 부트
스프링을 편리하게 사용할 수 있도록 지원
tomcat 같은 웹 서버를 내장해서 별도의 웹 서버를 설치하지 않아도됨

선택
스프링 데이터,세션 등 


스프링의 핵심 개념
스프링은 자바 언어 기반의 프레임워크
자바 언어의 가장 큰 특징은 객체 지향 언어
스프링은 객체 지향 언어가 가진 강력한 특징을 살려내는 프레임 워크

객체 지향
컴퓨터 프로그램을 명령어 목록으로 보는 시각에서 벗어나 여러개의 독립된 단위, 즉 객체들의 모이으로 파악하는 것 
각각의 객체는 메시지를 주고 받고 데이터를 처리할 수 있다.

다형성 

역할과 구현을 나눈다.  유연해지고 변경도 편리해진다.
클라이언트는 대상의 역할만 알면된다.
객체 설계시 역할을 먼저 부여하고, 그 역할을 수행하는 구현 객체를 만들면 된다.

DI 의존 관계 주입 DI 컨테이너
클라이언트의 교체 없이 기능을 확장할 수 있다.
