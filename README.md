# Backend Study Archive

백엔드 개발자로 성장하기 위해 Java, Spring Boot, Database, HTTP 등 백엔드 관련 지식을 학습하고 기록하는 저장소입니다.  
단순히 개념을 정리하는 데서 끝나지 않고, 예제 코드 작성, 문제 해결 과정, 트러블슈팅, 토이 프로젝트까지 연결하며 학습한 내용을 쌓아가고 있습니다.

## Goal
- 백엔드의 핵심 개념을 이해하고 직접 설명할 수 있기
- 단순 구현이 아니라 왜 이런 구조와 기술을 사용하는지 고민하기
- 학습한 내용을 코드와 프로젝트에 적용하며 실전 감각 키우기
- 꾸준한 기록을 통해 성장 과정을 남기기
